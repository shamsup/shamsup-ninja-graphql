# Ninja CMS

A GraphQL-based CMS built on Node.js

## Features
- [ ] Pages
- [ ] Posts
  - [ ] Custom Post Types
    - [ ] Images
    - [ ] Videos
    - [ ] Other Embeds
- [ ] Authentication
- [ ] Simple (but useful) Permissions System
  - [ ] Premade Roles
    - [ ] Admin
    - [ ] Author
    - [ ] Editor
    - [ ] User
  - [ ] Custom Roles
